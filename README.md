# 🌱 EcoReward

> **Rewarding Sustainability, One Donation at a Time.**

EcoReward is a modern web platform that encourages sustainable living by allowing users to donate reusable items instead of discarding them. The platform connects donors with verified NGOs and charitable organizations while rewarding users for making environmentally responsible choices.

By transforming unwanted items into valuable resources, EcoReward reduces waste, promotes a circular economy, and creates a positive environmental and social impact.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/yourusername/EcoReward.git
```

Navigate to the project directory

```bash
cd EcoReward
```

Install dependencies

```bash
npm install
```

Create an environment file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

# 🏗️ Build for Production

Build the project

# 🌱 Vision

EcoReward aims to build a more sustainable future by making item donations simple, transparent, and rewarding. Instead of allowing reusable items to become waste, the platform ensures they reach individuals and organizations that can benefit from them.

Every donation contributes to reducing landfill waste, conserving resources, and creating a cleaner environment.
